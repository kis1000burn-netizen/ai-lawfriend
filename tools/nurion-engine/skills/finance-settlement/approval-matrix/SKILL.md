# approval-matrix 스킬

## 목적
지급 금액과 종류에 따른 승인 단계를 강제합니다.  
누리온은 **승인 요청 생성·이력 저장까지만** 담당합니다.

## 승인 매트릭스
| 금액 | 필요 승인 |
|------|-----------|
| 0 ~ 100,000원 | 담당자 1인 |
| 100,001 ~ 1,000,000원 | 운영자 + 재무 |
| 1,000,001원 초과 | 대표 또는 지정 임원 |
| 예외 지급·수동 계좌 변경·환불 | 무조건 2인 승인 |

## 절대 금지
❌ 미승인 지급 실행  
❌ 승인 없이 계좌번호 변경  
❌ 승인 레벨 자동 우회
