# media-review 스킬

## 목적
승인 전 미디어(이미지·동영상·메뉴 사진)가 공개 노출되지 않도록 차단하고  
승인 이력을 기록합니다.

## 언제 사용하는가
- 업체 메뉴 사진, 이벤트 배너, 상품 이미지 등 미디어 업로드가 있는 플랫폼  
- 미디어 승인 워크플로우(업체 업로드 → 검수 → 승인 → 노출) 운영 시

## 판단 기준
| 조건 | 결과 |
|------|------|
| 미승인 미디어가 공개 URL에서 200 응답 | FAIL — critical |
| 미디어 승인 이력 파일 없음 | WARN |
| 승인 이력에 reviewedBy 없는 항목 | WARN |

## 자동조치 (actions.mjs)
| 조치 | 허용 여부 |
|------|-----------|
| 미노출 플래그 설정(`MEDIA_HIDDEN=1`) | ✅ apply 모드에서 허용 |
| 검수 이벤트 기록 | ✅ 허용 |
| 알림(notify) | ✅ 항상 허용 |
| 미디어 파일 삭제 | ❌ AI 실행 금지 |
| 승인 처리 | ❌ AI 실행 금지 |
